# PetPal Requirements

## Functional Requirements
1. User authentication (email/password)
2. User profile with multiple pets
3. Real-time one-to-one messaging
4. Friend system (send/accept/reject)
5. Pet matching (browse nearby pets, send interest)

## Non-Functional Requirements
- Cross-platform: Web, Android, iOS
- Backend: Node.js + Express + PostgreSQL + Socket.io
- Ubuntu development compatible
- No e-commerce, bookings, grooming, sitters, food, medicine
